📊 Analyse de la performance académique (2022–2023)

Description

Ce projet analyse les performances académiques des étudiants entre 2022 et 2023 à l’aide d’un tableau de bord interactif réalisé avec Microsoft Power BI.

L’objectif est d’identifier les tendances de réussite, détecter les difficultés académiques et proposer des recommandations pour améliorer les résultats des étudiants.

⸻

Indicateurs analysés
	•	Nombre total d’étudiants
	•	Taux de réussite
	•	Nombre d’étudiants admis
	•	Notes par étudiant et par matière
	•	Étudiants sans note (absences)
	•	Classement des étudiants
Outils utilisés
	•	Microsoft Power BI
	•	Microsoft SQL Server Management Studio
	•	SQL
